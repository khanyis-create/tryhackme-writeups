**TryHackMe — Search Skills | Room Writeup**

This room focused on one of the most underrated but essential skills in cybersecurity which is knowing how to find reliable, accurate information efficiently.

**Evaluating Information Sources**
Not everything on the internet is trustworthy. Good analysts evaluate sources based on four criteria: the credibility of the source, whether claims are backed by evidence and reasoning, whether information is presented objectively without bias, and whether multiple independent sources agree on the same facts.

**Search Engine Operators**
Advanced Google search operators help narrow down searches significantly. Key operators include using quotes for exact phrases, `site:` to limit results to a specific domain, `-` to exclude words, and `filetype:` to find specific file types such as PDFs or PowerPoint presentations.

**Specialized Search Engines**
Beyond Google, cybersecurity professionals use specialized tools such as Shodan to find internet-connected devices and servers, VirusTotal to scan files and hashes for malware, and Have I Been Pwned to check whether an email address has appeared in a data breach.

**Technical Documentation**
Official documentation from vendors like Microsoft, Apache, and Snort is the most reliable source for understanding how tools and commands work. I used this to look up Linux commands like `cat` (concatenate) and `ss` (socket statistics).

**Social Media as an OSINT Tool**
Social media platforms are valuable for Open Source Intelligence (OSINT). LinkedIn reveals professional backgrounds and technical skills, while platforms like Facebook can expose personal information that attackers might use to answer security questions and gain unauthorized access.

**Key Takeaway:** Knowing how to search effectively is not a soft skill, it's a core cybersecurity competency. Every investigation, threat hunt, and incident response starts with finding the right information from the right sources.
